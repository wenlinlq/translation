# 音频翻译应用

上传本地音频，自动完成 **语音识别 → 机器翻译 → 语音合成** 全流程，在网页中查看原文与译文，并下载文本与合成音频。

## 作品演示视频：https://b23.tv/vupaVQQ

## 核心功能

### 一键音频翻译流水线

上传音频后，系统自动依次调用百度智能云三项能力：

1. **语音识别（ASR）** — 将音频转为文字
2. **机器翻译（MT）** — 将原文翻译为目标语言
3. **语音合成（TTS）** — 将译文合成为目标语言 MP3

无需手动切换步骤，一次点击即可完成整条链路。

### 长音频与短语音双模式识别

| 模式             | 适用场景         | 说明                                                 |
| ---------------- | ---------------- | ---------------------------------------------------- |
| **音频文件转写** | 数分钟至更长音频 | 音频先上传至百度 BOS，再异步识别；单文件最大约 500MB |
| **短语音识别**   | ≤ 60 秒音频      | 未配置 BOS 时自动回退，适合快速试用                  |

系统会根据 `.env` 中是否配置 BOS 自动选择识别方式，无需手动切换。

### 多格式音频支持

- 支持 **WAV、MP3、M4A、AAC** 等常见格式
- 服务端内置 **ffmpeg**，自动将音频转为 16kHz 单声道 WAV 后再识别
- 无需用户自行预处理音频

### 多语言互译

- **源语言**：中文、英语、日语、韩语
- **目标语言**：中文、英语（可扩展更多语种）
- 可在界面中自由组合源语言与目标语言

### 识别结果可修正、可重跑

识别原文在界面中以可编辑文本框展示。若 ASR 结果有误：

- 可直接手动修改原文
- 点击「**用修正原文重新翻译并合成**」，跳过 ASR，仅重新执行 MT + TTS
- 无需重新上传音频，节省时间与 API 调用

### 多种合成发音人

支持百度 TTS 多款发音人，可在处理前或修正后切换：

- 度小鹿（精品女声，推荐）
- 度嫣然 / 度泽言（大模型女声 / 男声）
- 度逍遥、度星河、度灵儿等臻品 / 精品音色
- 度丫丫、度小美等基础音色

### 在线试听与文件下载

- 合成完成后可在页面内 **在线播放** 译文 MP3，支持播放 / 暂停
- 一键下载：
  - `result-*.txt` — 原文 + 译文 + 语言信息与耗时
  - `result-*-tts.mp3` — 目标语言合成语音
- 服务端同时保留输出至 `backend/output/` 目录

### 实时运行日志与耗时统计

界面底部展示本次处理的运行日志，包括：

- 各阶段耗时（BOS 上传、ASR 创建 / 轮询、MT、TTS、总计）
- 识别模式（音频文件转写 / 短语音）、任务 ID、音频时长等元信息
- 完整日志另存于 `backend/logs/run-*.log`，便于排查问题

### Web 界面 + 可选移动端

- **Web 端**：`backend/public` 内置静态页面，启动服务后浏览器即可使用
- **uni-app 端**：`frontend/` 提供移动端界面（可选），功能与 Web 端一致

## 典型使用流程

1. 选择本地 **WAV / MP3 / M4A** 音频文件
2. 设置 **源语言**、**目标语言** 与 **合成发音人**
3. 点击「**开始处理**」，等待 ASR → MT → TTS 完成
4. 查看识别原文与译文；如有误，修正原文后点击「重新翻译并合成」
5. 在线试听合成语音，或下载 txt / mp3 文件

## 快速开始

### 1. 配置百度智能云密钥

1. 登录 [百度智能云控制台](https://console.bce.baidu.com/)
2. **语音技术** 创建应用 → 获取 `API Key`、`Secret Key`（ASR、TTS）
3. **机器翻译** 创建应用 → 获取 `APP ID`、`密钥`（MT）
4. （推荐）开通 **对象存储 BOS**，用于长音频「音频文件转写」

### 2. 配置环境变量

```bash
cd backend
copy .env.example .env
```

编辑 `backend/.env`，填入上述密钥；长音频识别还需配置 `BOS_*` 相关项。

### 3. 安装并启动

```bash
cd backend
npm install
npm start
```

浏览器打开：**http://localhost:3000**

## 注意事项

- 长音频识别需在 `.env` 中配置 `BOS_ACCESS_KEY_ID`、`BOS_SECRET_ACCESS_KEY`、`BOS_BUCKET` 等；未配置时仅支持 ≤ 60 秒短语音。
- 可通过 `BAIDU_AASR_PID` 切换转写模型（如中文 80006、英文 1737）。
- 请勿将 `.env` 提交到版本库。
